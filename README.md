# vaws

> Color formatting tool by Vaws for the console.

## Installing and using
Install using NPM:
```bash
$ npm i @vawsbot/colors
```
Usage:
```js
const VawsColors = require('@vawsbot/colors');
console.log(VawsColors.yellow('Yellow text')):
```

## All options
Yellow, cyan, green, red, underscore, black. -->
